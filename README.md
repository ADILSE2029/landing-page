# Seasons Landing Page

A single-page website built with HTML and CSS as part of The Odin Project's Foundations course, based on the Landing Page project.

## About

This page explores the four seasons, spring, summer, autumn, and winter, through a hero section, an informational season showcase, a quote section, and a call-to-action linking to further reading. In-page navigation links allow jumping directly to different sections of the page.

## What I Practiced

- Flexbox layout (rows, alignment, spacing)
- Building reusable section patterns (header, hero, info grid, quote, CTA)
- Centering content with max-width and margin auto
- In-page anchor navigation using id and href
- Structuring and styling a full page from scratch without a starter template
